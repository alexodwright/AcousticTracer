\section*{Project Proposal}

\subsection*{Introduction}
Simulate how sound travels through a 3D environment, and replay that simulation as a 3D visualisation in a web app.

Workflow:

\begin{figure}[h!]
    \includegraphics[width=0.7\textwidth]{../assets/images//workflow.jpg}
\end{figure}

\subsection{Desired Features}
    \begin{itemize}
        \item Use of genetic algorithms to determine the optimal source location
        \item Observer Sound Intensity as a Voxel Heatmap
        \item 3D Scene Renderer
        \item Acoustic raytracer and raycaster
        \item React front-end with user-uploaded \texttt{.glb}, (environment scan) files
    \end{itemize} 

\subsection{Implementation Stack and Technologies}
    \begin{description}
        \item [C] Performant raytracing and for genetic algorithm computation
        \item [React] Modern front-end and user interactivity
    \end{description}
